Anime Watchlist Progressive Web App created with [REACT](https://reactjs.org/).


## Features

* explore shows
* track over episodes 
* search and add 


## Development

Run the following commands in terminal to run on your local machine.

```bash 
git clone https://github.com/AnujSharma141/Daijobu.git
cd Daijobu
npm install
npm start
```
